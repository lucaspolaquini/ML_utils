# Aula 4 - Feature Scaling

Conteúdo desta pasta:
- `Normalização e padronização de dados - Feature Scaling.ipynb`
- `Churn_Modelling.csv`

## Sobre esta aula

Esta aula explora a importância do escalonamento de dados em problemas de machine learning, especialmente quando modelos são sensíveis à distância entre observações.

## Pontos principais

- Diferença entre normalização (min-max scaling) e padronização (z-score).
- Por que as escalas diferentes podem afetar modelos de machine learning.
- Exemplo prático de churn prediction com `Churn_Modelling.csv`.
- Análise de distribuição das variáveis usando boxplots.
- Comparação de desempenho de modelos com e sem escalonamento.

## Como usar

1. Abra `Normalização e padronização de dados - Feature Scaling.ipynb`.
2. Execute o notebook e acompanhe a preparação dos dados, a transformação das features e a modelagem.
