# Módulo de Machine Learning

Este módulo reúne as principais aulas de Machine Learning com foco em regressão, pré-processamento e redução de dimensionalidade.

## Estrutura deste módulo

- `Aula 1 - Conceitos básicos e aplicações`
  - Material teórico e conceitual sobre IA e ML.
- `Aula 2 - Regressão linear e métricas de validação`
  - Prática com regressão linear simples e caso end-to-end de regressão múltipla.
- `Aula 3 - Redução de dimensionalidade`
  - Introdução ao PCA e à importância da padronização de variáveis.
- `Aula 4 - Feature Scaling`
  - Normalização e padronização aplicadas a um caso de churn.

## Pontos-chave do módulo

- Regressão linear como técnica básica para previsão de valores numéricos.
- Avaliação de modelos com métricas como MSE, MAE e R².
- Importância da visualização dos dados e análise de correlação.
- Como lidar com dados faltantes, codificação e pipelines em um fluxo de machine learning.
- Redução de dimensionalidade com PCA para capturar variabilidade essencial dos dados.
- Por que escalar variáveis é fundamental para modelos sensíveis à distância.
- Diferença entre normalização (min-max) e padronização (z-score).

## Instruções rápidas

1. Ative o ambiente virtual se existir: `source .venv/bin/activate`
2. Instale dependências: `pip install -r requirements.txt`
3. Abra os notebooks nas pastas de cada aula e execute as células na ordem apresentada.

## Observações

- O arquivo `requirements.txt` contém as dependências usadas nos notebooks.
- O `.gitignore` já está configurado para ignorar ambientes virtuais, arquivos gerados pelo Python e checkpoints do Jupyter.
