# Aula 1 - Conceitos básicos e aplicações

Conteúdo desta pasta:

## Sobre esta aula

Esta aula apresenta os conceitos fundamentais de inteligência artificial e machine learning. Ela traz uma visão geral dos principais termos, aplicações e do fluxo de trabalho típico em projetos de dados.

## Objetivos

- Entender o que é inteligência artificial e machine learning.
- Conhecer as etapas de um projeto de dados.
- Ver exemplos de aplicações práticas.

## Como usar

Abra o PDF e estude os conceitos antes de avançar para as aulas práticas de regressão e pré-processamento.
