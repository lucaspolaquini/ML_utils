# Aula 3 - Redução de dimensionalidade

Conteúdo desta pasta:
- `Case_PCA_(Principal_Component_Analysis).ipynb`
- `players_22.csv`

## Sobre esta aula

Esta aula apresenta a técnica de Análise de Componentes Principais (PCA), utilizada para reduzir a dimensionalidade de conjuntos de dados e identificar as variáveis que carregam a maior parte da variância.

## Pontos principais

- O que é PCA e por que usamos redução de dimensionalidade.
- Impacto das escalas das variáveis em análises multivariadas.
- Necessidade de padronização antes de aplicar PCA.
- Aplicação prática com o dataset `iris` do `scikit-learn`.
- Análise de componentes principais e de variância explicada.

## Como usar

1. Abra `Case_PCA_(Principal_Component_Analysis).ipynb`.
2. Execute o notebook para ver o fluxo de pré-processamento e aplicação do PCA.
3. Use `players_22.csv` como dataset adicional para praticar a técnica se desejar.
