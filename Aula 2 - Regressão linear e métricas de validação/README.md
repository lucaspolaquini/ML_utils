# Aula 2 - Regressão linear e métricas de validação

Conteúdo desta pasta:
- `Regressão Linear Simples.ipynb`
- `Case end to end Regressão.ipynb`
- `Sorvete.xlsx`
- `housing.csv`

## Sobre esta aula

Esta aula foca em regressão linear e nas métricas de avaliação de modelos de regressão. Ela apresenta um estudo de caso simples com vendas de sorvete e um fluxo completo de regressão múltipla usando dados de imóveis.

## Pontos principais

- Regressão linear simples para prever vendas com base na temperatura.
- Visualização dos dados com scatter plots.
- Cálculo de correlação entre variáveis.
- Separação de dados em treino e teste.
- Avaliação de desempenho com MSE, MAE e R².
- Caso end-to-end com dados de imóveis:
  - Leitura de `housing.csv`.
  - Preprocessamento de dados.
  - Imputação de valores faltantes.
  - Codificação de variáveis categóricas.
  - Construção de pipeline com `scikit-learn`.
  - Comparação entre modelos de regressão linear e árvore de decisão.

## Como usar

1. Abra `Regressão Linear Simples.ipynb` e execute o notebook do início ao fim.
2. Em seguida, abra `Case end to end Regressão.ipynb` para ver um processo completo de modelagem.
